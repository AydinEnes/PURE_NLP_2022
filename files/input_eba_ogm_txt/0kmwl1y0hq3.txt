BiYOLOJi MiTOTiK EVRE

 
Mitotik evre iki süreçten oluşur:

1. Çekirdek bölünmesi (karyokinez / mitoz)

2. Sitoplazma bölünmesi (sitokinez)

   

Profaz

O Kromatin iplikler sıkılaşarak daha
belirgin hale gelir.

O Sentrozomlar iğ ipliklerini
oluşturarak kutuplara doğru hareket
eder. (Hayvan hücresi)

O Çekirdek zarı parçalanır,
çekirdekçik kaybolur.

Metafaz

O Kromozomlar ekvatoral düzlemde
tek sıra halinde dizilmişlerdir.

O Kromozomların en net görüldüğü
evredir.

Anafaz
0 Kardeş kromatitler birbirinden

ayrılır.
0 iki kutupta eşit sayıda kromozom
gözlenir.

Telofaz
O Kromozomlar kromatin ağına
dönüşmeye başlar.
0 Çekirdek zarı yeniden oluşur.
0 iğ iplikleri kaybolmaya başlar.

2. Sitoplazma Bölünmesi (Sitokinez)
0 Hayvan hücrelerinde
boğumlanma ile Sitoplazma bölünür.
Boğumlanmada mikrofilamentler
hücreyi ikiye ayırır.
0 Bitki hücrelerinde ara Iamel ile

Hücre Döngüsünde Kromozom Sayısı Değişimi

Sitoplazma bölünür. Golgi tarafından
oluşturulan ara lamel hücreyi ikiye
ayırır.

Mitozun Genel özellikleri

0 Bir hücreden iki yeni hücre
oluşur.

0 Farklı kromozom takımlarına
sahip (n, 2n,3n) hücrelerde de mitoz
gerçekleşebilir.

O Bölünme sonucunda kromozom
sayısı sabit kalır.

0 Genetik çeşitlilik görülmez.

O Oluşan hücrelerin büyüklüğü,
organel sayısı, Sitoplazma miktarı
farklılık gösterebilir.
